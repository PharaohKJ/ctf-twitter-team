# Civic Tech Forum Twitter 実況チーム ドキュメント

このドキュメントはCivic Tech Forum運営チームのひとつ、twitter実況チームのためのドキュメントです。

## 狙い

Civic Tech Forum Twitter 実況チームは、会の内容をできるだけ

- 臨場感あるよう写真付きでリアルタイム
- togetter でまとめやすいように
- togetter　で読み返したら流れがわかるように
- (絶対に) ハッシュタグつきで

実況ツイートし

- 本会の盛況ぶり
- 来場してない人にもツイートを読んだら参加した気持ちになれる
- 来年こそは参加したくなる
- 参加したひとは思い出せる

ということを狙う。

以降、実況ツイートといったら上記をできるだけ達成できるようにツイートすることを指す。

## スタッフ

便宜上

- 運営スタッフ
- ヘルプスタッフ

に分ける。前者は事前準備含めて担当する。後者は当日手伝っていただくサポートスタッフのことを指す。

スタッフは

- 実況してもいい個人アカウントを持っている
- 10分のLTを写真付き実況できる
- 実況ツイートを読んだことがある
- togetter を読んだことがある
- ハッシュタグで検索できる

は必須

- 実況ツイートを読んだことがある or したことがある
- togetter を読んだことがある or まとめたことがある

が望ましい。特にタイピングが苦手、twitterに手慣れていない場合は担当するのが難しいので自信の無い方は申し出ること。
（割とハードルは高めです！）

## やること

自分が担当しているパートの本ハッシュタグ + パート用ハッシュタグを付けて実況ツイートします。

## 書式

- ツイート支援ツール ( https://tewaketter.herokuapp.com ) を用意するのでそれを使って書式を確認してください。
  - どの部屋のツイートなのかわかるようにしてください。
  - `【CTF2019｜#カフェ1】 #civictechjp` のように定型文を挿入することを支援するツールになります
- 発表者の言葉なのか、実況担当の感想なのかわかるようにしてください。
- ハッシュタグを忘れずに！

### 流れ


ツイート実況席を用意していただいているのでそこに着席して準備。

LT登壇者が準備中に、支援ツールにアクセスして、「登壇者名」「発表タイトル」「パートの名前」などが揃ったテンプレートを用意する。

**公式アカウントで** 「登壇者名」「発表タイトル」「パートの名前」を文字で + ←がわかるような構図の写真を撮影し、添付してツイート。前述した書式や、支援ツールを活用してください。

例) https://twitter.com/civictechforum/status/1002767795426168833

これを終えたら **スタッフ自身のアカウント** で、本ハッシュタグ + パート用ハッシュタグを付けてどんどん実況ツイート。(公式アカウントを使うと制限に引っかかってしまうことがあるため、分散させるためにこのようにします)

- 準備の雰囲気を実況ツイート
- でてくる印象深い言葉やまとめの言葉を捕まえてツイート
  - どのような雰囲気、表情でもわかるよう写真もあるとなおよい
- パンチラインのスライドの写真

LT終了まで繰り返す。

最後にtogetterまとめを作成する(後述)。

### togetter流れ

自分のパートについて実況を終えたら、以下の手順で各自まとめてください。

1. PCで [まとめよう、あつまろう - Togetter](https://togetter.com/) にアクセスし、公式アカウントでtwitterログインする。
  - PCない方は運営スタッフまで連絡する
2. 右上の「まとめる」ボタンを押す
3. **作成したまとめのタイトルを入力** は上記より引用する。
4. **作成したまとめの簡単な解説文を入力** は前述したとりあえずテンプレート文を入力しておく。
  - 運営スタッフがあとで作成します。運営スタッフは他のまとめのリンク一覧を作成すること。  
5. **カテゴリーの選択** は 【ログ・日記】 を設定。
6. **設定したいタグ** は「イベント」と入力する
7. **他のユーザによる操作の制限** まとめ編集を許可のみONへ。
8. **公開設定** はいったん 限定公開（非公開） > リンクを知っている人限定 にしておくこと。
9. 左下の検索欄に自分のパートのハッシュタグを指定してツイートを検索し、担当したツイートや、それに付随するツイートを探しだす。
  - **RTを除く** はONで。
  - 全体タグしかつけない方もいらっしゃるので、なるべく探す。
  - 文章の雰囲気から関連しているかどうかを推測するしかないのでできるだけがんばるしかない。
  - 他のツイートが混ざる、被るしてしまっても消して対応しますので、いったんは **モレなく** を重視してください
10. 探しだしたら **移動** をクリックして右側のまとめに追加していく
11. 右側まとめの最下段に **↓時間順** というボタンがあるのでそれを押して時系列にざっくり並べる。
12. まとめ終わったら **重複分を削除** を押し **下書き保存** をクリックして保存する
13. **投稿する** をクリックする
14. URL を運営に連絡する

まで行ってください。このあとは運営担当者

1. もらったtogetter URLにアクセスし、内容を精査
2. **作成したまとめの簡単な解説文を入力** に、 **TODO** と記載
3. **編集可能** を外す
4. 全体公開する

会が終わったら

1. もらったtogetter URLらにアクセスし、内容を精査
2. もうできている全部のリンク一覧を記載
3. 保存して全体公開しなおす


後日（みんなが↑のツイートを読んで感想とかもツイートしてくれたあたりで

1. もらったtogetter URLらにアクセスし、内容を精査
2. 左下の検索欄にそのパートのハッシュタグを指定してツイートを検索し、担当したツイートや、それに付随するツイートを探しだす。
  - **RTを除く** はONで。
  - 文章の雰囲気から関連しているかどうかを推測するしかないのでできるだけがんばるしかない。
3. 時系列に並んでいるツイートを上から順番に流れがわかるようにマウスで並び替える。
  - 他のツイートが混ざる、被るしてしまっても消して対応しますので、いったんは **モレなく** を重視してください
  - リアルタイムに言及していない発言もあるので、それもなるべくがんばって推測して並び替える。
4. 保存して全体公開しなおす
